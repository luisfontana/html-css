De mesma forma o uso do ITÁLICO destacado na frase de forma correta não só como alteração de forma mas tbm com significado semântico! (usando a tag EM para isso)

Marcação de texto

Texto deletado

O texto deletado serve para espresar que determinada parte do que foi escrito não é inportante. Como essa a partir do ponto!

Texto inserido

O texto inserido é o exato oposto ao texto deletado. Veja se um refere-se a uma parte sem importância o outro menciona partes de maior importância, de forma semântica.

Tambem pode ser usado para sublinhar texto a tag U mas de forma não semântica. Como foi feito neste parágrafo.

Texto grande e pequeno

Caso for necessario fazer que uma parte do texto estege um pouco maior que o restante pode ser usado a tag (big) mesmo que ela ja estaje em desuso. Da mesma forma para deixar uma parte do texto menor como letras pequenas de um contrato, para isso usamos a tag (small) essa ainda é uma tag válida.

Texto sobrescrito e subescrito

O texto sobrescrito seria como escrever a formula do Bario da tabela periódica, que seria ( Ba56 ). Tag usada SUP.

O texto subescrito é melhor explicado pela fórmula da água. Vulga (H2O). Tag usada SUB.
